# University Management System

A web application built using ASP.NET Core MVC as part of the ITI Scholarship.

## Features

- Authentication & Authorization
- ASP.NET Identity
- Role-Based Access Control
- CRUD Operations
- Entity Framework Core
- Repository Pattern
- Unit of Work Pattern
- Dependency Injection
- 3-Tier Architecture
- Front-End & Back-End Validation
- Custom & Built-in Middleware

## Technologies Used

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- ASP.NET Identity
- Bootstrap
- HTML/CSS/JavaScript

## Architecture

- Presentation Layer
- Business Logic Layer
- Data Access Layer

## Database Relationships

- One-to-Many
- Many-to-Many
